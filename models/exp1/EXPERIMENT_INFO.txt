EXPERIMENT 1: Baseline Random Forest Models
================================================================================

OBJECTIVE
---------
Establish baseline performance for PM2.5 and PM10 prediction using basic
Random Forest models with default hyperparameters.

APPROACH
--------
Model: Random Forest Regressor
Features: 23 predictors
  - Satellite: NO2, AOD
  - Weather: T2M, D2M, U10, V10, SP, BLH
  - Derived: WindSpeed, WindDirection, RelativeHumidity, TempCelsius
  - Temporal: DayOfYear, Month, Season, DayOfWeek, Week, IsWeekend
  - Interactions: AOD_Temp, NO2_AOD
  - Location: Latitude, Longitude

Hyperparameters:
  - n_estimators: 200
  - max_depth: 20
  - min_samples_split: 5
  - min_samples_leaf: 2
  - No hyperparameter tuning applied

Data Split:
  - Training: 15,288 samples (70%)
  - Validation: 3,276 samples (15%)
  - Test: 3,276 samples (15%)
  - Total: 21,840 daily station measurements (2022-2024)

RESULTS
-------
PM2.5 Model:
  - Test R²: 0.4490
  - Test RMSE: 16.89 µg/m³
  - Test MAE: 10.77 µg/m³

PM10 Model:
  - Test R²: 0.5471
  - Test RMSE: 48.67 µg/m³
  - Test MAE: 31.07 µg/m³

TOP FEATURES (PM2.5):
  1. T2M (Temperature)
  2. TempCelsius
  3. Latitude
  
TOP FEATURES (PM10):
  1. AOD_Temp interaction
  2. Longitude
  3. DayOfYear

ASSESSMENT
----------
✓ Reasonable baseline performance for air quality prediction
✓ R² values (0.45-0.55) typical for satellite-based PM models
✓ Feature importance makes physical sense
• Room for improvement through hyperparameter tuning and algorithm upgrades
